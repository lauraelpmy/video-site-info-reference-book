# video-site-info-reference-book
该仓库作为影视与视频类网站的信息参考集合，对站点名称、访问地址及内容定位进行统一归档。项目目标在于形成稳定、可更新的信息记录，方便进行横向对比与历史追踪。仓库不存储任何影视资源，也不提供在线播放服务，仅用于资料整理与研究分析，所有内容均来源于公开网络。
